
# Parking

This API REST was generate in Node.Js


## Run Locally

Clone the project

```bash
  git clone https://github.com/JOHNWILLIAMREYES/parking.git
```

Go to the project directory

```bash
  cd parking
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## API Reference

#### Base API 

```Base API Url
  http://localhost:3002/api/
```
